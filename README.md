# k6-LoadTests-Reports-AzureDevOps_VotacaoPesquisaInteresses
Exemplo de implementação de testes de carga em JavaScript para execução com a ferramenta k6, incluindo a geração de relatórios com os resultados através do projeto k6-reporter. Este repositório também inclui um Pipeline do Azure DevOps para a execução automatizada dos testes de carga de um site com uma pesquisa sobre interesses pessoais.

Projeto que serviu de base para a implementação deste exemplo (imagem pública no Docker Hub -> **renatogroffe/sitepesquisainteresses-eventhubs-appinsightsconnstring-dotnet6**):
